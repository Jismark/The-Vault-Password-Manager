The Vault Password Manager is a local desktop application.

## Data Collection
The application does not collect, store, or transmit any personal data to the developer or any third party.

## Local Storage
All user data (passwords, notes, and settings) are stored locally on the user's device in encrypted form.

## Google Drive Integration
If the user chooses to enable Google Drive synchronization, the application accesses the user's Google Drive solely to upload and download the encrypted vault file.
The application only accesses files created by the application itself.
No data is sent to the developer.

## Encryption
All vault data is encrypted locally before being stored or synchronized.
The developer cannot access or decrypt user data.

## Third Parties
No third-party analytics, tracking, or advertising services are used.
